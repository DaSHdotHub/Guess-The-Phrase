The idea will be to implement a quiz game, the set of data will be provided List of Objects that will contain:

  • The phrase where one word was exchanged
  • The context where the phrase is from, e.g. name of book or movie
  • The correct word
  • A list of wrong answers
  • The wrong word from the phrase

The Objectlist will be provided as a csv file inside the assets, on loadup it will be cached
